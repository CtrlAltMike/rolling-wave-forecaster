Rolling Wave Forecasting Tool

About This Tool

This tool uses Rolling Wave Forecastingâ€”a technique where near-term iterations are planned in detail (with Story Point forecasts), and longer-term iterations remain at a higher level. As actual performance data (delivered Story Points) becomes available for completed iterations, the tool calculates the average deviation (error) between the forecast and actual outcomes. This average error is then applied as an adjustment to the future iterations.

Inputs:

  • Iteration Name (modifiable text field)
  • Initial Forecast in Story Points (planned work for each iteration)
  • Actual in Story Points (delivered work for completed iterations)
Output:
  • Revised Forecast in Story Points, which adjusts the future forecasts based on past performance.
